Shoe Swap is a solely built app which allows users to buy and sell used shoes. A user logs in and is taken to a page which displays the shoes the user has sold, purchased, and currently is selling. There is a link to a page which displays all shoes for sale on the site. Users can purchase shoes from there.

TECHNOLOGIES

Ruby on Rails, PostgreSQL, Bootstrap, HTML5, CSS3
